About MovieFinder

Yelyzaveta Kareieva · S00267553

MovieFinder is a single-page Angular application that lets you search for movies and TV shows using the OMDb API. Browse results, and click any title to view its full details.

🔍 Search

Search any movie or TV show by title. Results are pulled live from the OMDb database.

🎬 Details

View full details including plot, cast, director, IMDb rating, and box office figures.

⚡ Built with Angular

Uses Angular 19 signals, standalone components, and the Angular Router for navigation.

🌐 OMDb API

Data is sourced from the Open Movie Database, which indexes thousands of titles worldwide.

Built as part of a college assignment · Angular · Bootstrap · TypeScript
